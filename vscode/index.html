# Visual Studio Code 权威指南

# 插件类型

  • Azure 云计算插件
  • Debuggers 调试器插件
  • Extension Packs 插件包，包含多个插件
  • Formatters 代码格式化插件
  • Keymaps 快捷键映射插件
  • Language Packs 语言插件
  • Linters 静态检查插件
  • Programming Languages 编程语言插件
  • SCM Providers 源代码控制管理器插件
  • Snippets 代码片段插件
  • Themes 颜色主题与图标主题插件
  • Others 其他类型插件

# 常用插件

  • Code Runner 运行多种语言的代码片段或代码文件
  • REST Client 测试 REST API
  • Bracket Pair Colorizer 2 为代码中结对出现的括号提供颜色高亮功能
  • indent-rainbow 为代码中的缩进提供颜色区分
  • vscode-pigments 为 16 进制的 css color 代码提供背景颜色
  • Peacock 为每一个 vscode 窗口提供一种颜色
  • GitHub Pull requests 可以轻松的审查和管理 GitHub 上的 Pull requests
  • Git Graph 图形化界面，可以显示各类信息也可以直接进行许多操作
  • GitLens 更好的回顾代码历史，以及如何演变，比如更改代码的人员、原因、时间，比如 GitLens: Open Settings 查看配置
  • EditorConfig for VS Code 统一编码样式
  • Code Spell Checker 为代码单词进行拼写检查，并提供自动修复功能。
  • Better Comments 将代码注释分为警告、高亮、待办事项等，提供颜色区分
  • Image preview 可以通过鼠标浮动代码中的 url，支持实时预览
  • Todo Tree 将工作区所有 TODO、FIXME 标签内容显示在树状视图中
  • TODO Highlight 为 TODO、FIXME 标签提供高亮功能
  • Bookmarks 可以为任何一行代码提供标签跳转
  • CodeSnap 将代码导出为图片格式
  • WakaTime 可以记录开发者日常编写代码的使用情况，包括时间、项目、语言等
  • Paste JSON as Code 可以把 JSON 或 TypeScript 转成其他语言，也可以 JSON => TS 类型声明文件
  • LeetCode 程序员 offer 收割利器
  • 主题插件 - Custom CSS and JS Loader 自定义 JS 和 CSS 插件
  • 主题插件 - SynthWave '84 霓虹灯发光风格
  • 主题插件 - Vibrancy 毛玻璃效果
  • 劳逸结合 - 搜索鼓励师，写代码不再孤单

# 语言插件

  • ESLint
  • StyleLint
  • Jest
  • Impost Cost 内联显示导入包的大小
  • Turbo Console Log 通过快捷键自动帮你写 console.log
  • Vetur 支持 Vue 2 风格代码语法高亮，自动补全等
  • Vue VS Code Extension Pack 热门的 Vue 插件包
  • Prettier - Code formatter 热门的代码格式化插件

# 其他插件

  • Auto Rename Tag 自动重新命名标签
  • CSS Peek 快速查看 CSS 定义，还可以在 CSS 内联编辑器中直接修改
  • Element UI Snippets 代码片段
  • Highlight Matching Tag 标签高亮
  • Parser JSDoc 自动添加 JSDoc 注释
  • vscode-icons 为文件添加图标
  • vscode-pangu 在中英文之间添加空格，同时去掉了一些末尾多余的空格
  • Volar 专门为 Vue 3 构建的语言支持插件
